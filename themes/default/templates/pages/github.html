Projetcs on github

You may find me on [github](https://github.com/mklabs).

nabe

Wheat-like blog engine. Most inspiration comes from Wheat. Built right on top of Connect, it can use git to provide some neat stuff like search (git grep) and access to post revisions. Also have a tiny admin backend where one could edit/add post (a save is then a git commit, with its commit message).

backnode

Latest crazy experiments. The goal is to let me use most of the fantastic Backbone library on the server-side. There are already several really good tutorials on this, namely to share Backbone.Models. This experiment is designed and implemented as a connect middleware and allow one to use Backbone.Router to handle incoming request, and Backbone.Views to drive the output of a template file.
